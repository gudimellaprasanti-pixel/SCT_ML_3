# Cats vs Dogs Image Classification using SVM

Implement a Support Vector Machine (SVM) to classify images of cats and dogs.

## Dataset
Microsoft Cats and Dogs Dataset (PetImages)

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-Learn
- Matplotlib
- Google Colab

## Steps Performed
1. Loaded Cat and Dog images.
2. Resized images to 64×64 pixels.
3. Converted images to grayscale.
4. Flattened images into feature vectors.
5. Split dataset into training and testing sets.
6. Trained an SVM classifier.
7. Evaluated model performance.
8. Displayed actual images with predicted labels.

## Results

Accuracy Achieved: 49.5%

The model successfully classified cat and dog images and displayed predictions alongside actual images.
